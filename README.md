# EasyShop

## Description of the Project

EasyShop is the back end API for an online store. It handles filtering through products by genre, color, and price.
The user is able to add to cart and clear their cart. The user can manage their profile information, and it will save what the user updated to the database.

## User Stories
- As a customer I want to browse all categories and see products within them so that I can find the items I want to buy.
- As an admin I want to add, change, or remove categories so the stores department list stays accurate.
- As a user I want to see all the items I've added to the cart so that I can review my order.
- As a user I want to add items to the cart so that the item is saved to my account.
- As a user I want to change the quantity of an item in my cart so that I can buy more than one.
- As a user I want to clear my whole cart so that I can start fresh.
- As a user I want to see and change my profile information so that my account can stay up to date.

### Running the Application in IntelliJ

Follow these steps to get your application running within IntelliJ IDEA:

1. Open IntelliJ IDEA.
2. Select "Open" and navigate to the directory where you cloned or downloaded the project.
3. After the project opens, wait for IntelliJ to index the files and set up the project.
4. Find the main class with the `public static void main(String[] args)` method.
5. Right-click on the file and select 'Run 'ECommerceApplication.main()'' to start the application.
